# Screen application

## Description

Permet de faire tourner des applications en background et se déconnecter.
Par exemple, screen permet de se logger sur un rapsberry pi en SSH, lancer un traitement, puis le laisse en tache de fond et quitter le shell sans intérrompre le traitement.

## Common usage

### Start a session

`screen -S nom_de_la_session`

### Quit a session

Type `Ctrl+A d`

### Restore  a sessions

`screen -r nom_de_la_session`
